Haiku
=====

Haiku (meaning a very short form of [Japanese poetry](http://en.wikipedia.org/wiki/Haiku)) is a random name generator.

## Install

    go get github.com/nwjlyons/haiku

This is an example of what happens when you run `haiku`:


    haiku
    autumn-field
